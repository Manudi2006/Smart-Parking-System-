## Overview
Parkers is a smart parking solution designed to reduce congestion and improve user experience in shopping complexes.

## Key Features
- Real-time parking availability
- QR-based entry system
- Digital payments
- Vehicle tracking (Find My Vehicle)
- ANPR verification
- EV-friendly parking options
- Guest mode access
- Community-based spot sharing

## Purpose
This academic project involved problem identification, market analysis, and solution design. The system was developed to address parking inefficiencies in shopping malls through innovative and technology-based improvements.
